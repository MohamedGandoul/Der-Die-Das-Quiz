## 🌟 What is This?



Welcome to **Der Die Das Quiz** - your fun way to master German articles! 🇩🇪✨



Practice German nouns and choose the correct article: **der, die or das** before time runs out!



This game helps you learn German in a simple, fast and addictive way 🎯



<div align="center">
